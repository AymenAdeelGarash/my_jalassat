# دليل إعدادات متجر جلسات شتوية ⚙️

يتحكم ملف `data/settings.json` في الخصائص الأساسية للمتجر. يمكنك تعديل هذا الملف مباشرة أو عبر لوحة التحكم.

## شرح المتغيرات

| الإعداد | الوصف | القيم المقبولة | مثال |
| :--- | :--- | :--- | :--- |
| `store_name` | اسم المتجر الذي يظهر في العنوان والفوتر | نص | `"جلسات شتوية"` |
| `store_phone` | رقم الواتساب لاستقبال الحجوزات | رقم دولي (بدون مسافات) | `"+966561365655"` |
| `store_status` | حالة الموقع (مفتوح أو مغلق للصيانة) | `"open"` أو `"closed"` | `"open"` |
| `instagram` | حساب الانستقرام | اسم الحساب | `"@winter_sessions"` |
| `snapchat` | حساب السناب شات | اسم الحساب | `"@winter_sessions"` |
| `primary_color` | اللون الأساسي للموقع (الأزرار، العناوين) | كود لون HEX | `"#c9a050"` (ذهبي) |
| `snow_enabled` | تفعيل أو إيقاف تساقط الثلج | `true` (مفعّل) أو `false` (معطّل) | `false` |

## كيفية التعديل

### الطريقة الأولى: عبر لوحة التحكم (الأسهل)
1. اذهب إلى الرابط: `http://localhost/galassat/master-control`
2. انتقل إلى قسم **الإعدادات العامة**.
3. عدل القيم ثم اضغط **حفظ الإعدادات**.

### الطريقة الثانية: تعديل الملف مباشرة
1. افتح الملف `data/settings.json`.
2. غيّر القيم بين علامات التنصيص `""`.
3. **تنبيه:** لا تحذف الفواصل `,` بنهاية السطور، وتأكد من كتابة `true` أو `false` بدون علامات تنصيص.
